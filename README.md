>  Nesse desafio criado em `Expo` para reforçar os **conceitos mais importantes** do `React Native`, porém aplicando três pontos principais no modulo como:

- Navegação com `React Navigation`
- Armazenamento local com `AsyncStorage`
- Estilização com `Styled Components`
######
# Sobre o desafio

### Nesse desafio, foi desenvolvido uma aplicação de cadastro de refeições, que contém as seguintes funcionalidades:

- ✅ Adicionar uma nova refeição
- ✅ Editar uma refeição
- ✅ Remover uma refeição da listagem
- ✅ Mostrar as estatísticas do progresso da dieta
- ✅ Navegação entre telas em pilha
- ✅ Armazenamento local das refeições

### conceitos usados como:

- ✅ Estados
- ✅ Imutabilidade do estado
- ✅ Listas e chaves no React Native
- ✅ Propriedades
- ✅ Componentização
- ✅ Parâmetros de navegação

 #### Além disso, para completar esse desafio foi necessário aprender sobre conceitos  de listas divididas em seções que podem ser mais facilmente implementadas com a <a href="https://reactnative.dev/docs/sectionlist" style='color:#77cff2' >SectionList</a> do que com a <a href="https://reactnative.dev/docs/flatlist" style='color:#77cff2' >FlatList</a>.

 # Intruções

 - Clona o projeto na maquina local
 - Entra na pasta do projeto e executa ***npm install*** ou ***yarn*** para fazer download e instar as dependências do projeto. 
 - Rodar **npx expo start**
 - Escolher o emulador
